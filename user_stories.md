# User Stories - Smart Care Clinic Management System

### 1. Admin User Story
**As an Admin**, I want to **add and manage doctor profiles** in the system, so that **the clinic registry is always accurate and up to date**.
* **Acceptance Criteria:** * Ability to input doctor name, specialty, and contact details.
    * Data is saved to the MySQL database.

### 2. Patient User Story
**As a Patient**, I want to **search for doctors by their specialty**, so that **I can find the right healthcare provider for my specific needs**.
* **Acceptance Criteria:**
    * Search bar filters results based on "Specialty" (e.g., Cardiologist).
    * Results display doctor availability.

### 3. Doctor User Story
**As a Doctor**, I want to **view my list of scheduled appointments**, so that **I can prepare for my patients' visits in advance**.
* **Acceptance Criteria:**
    * Access to a dashboard showing patient names and appointment times.
    * Ability to view associated patient records.
